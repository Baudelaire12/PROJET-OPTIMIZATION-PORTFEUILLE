Optimisation de Portefeuille d'Investissement

Un projet combinant la Théorie Moderne du Portefeuille et l'apprentissage automatique pour optimiser les investissements

Voir sur GitHub

À propos du projet

Ce projet vise à optimiser un portefeuille d'investissement en combinant la Théorie Moderne du Portefeuille (MPT) de Markowitz et des modèles d'apprentissage automatique. Il utilise des données historiques de Yahoo Finance pour prédire les rendements et minimiser les risques.

Fonctionnalités principales

Collecte de données réelles

Récupération de données financières via l'API Yahoo Finance avec gestion robuste des erreurs et des données manquantes.

Optimisation de portefeuille

Implémentation de l'algorithme d'optimisation de Markowitz pour trouver le portefeuille optimal sur la frontière efficiente.

Modèles d'apprentissage automatique

Utilisation de modèles ML pour prédire les rendements futurs et améliorer l'allocation d'actifs.

Tableau de bord interactif

Interface utilisateur interactive avec Streamlit pour visualiser et analyser les résultats d'optimisation.

Comment démarrer

Pour commencer à utiliser ce projet :

  1. Clonez le dépôt GitHub
  2. Installez les dépendances avec pip install -r requirements.txt
  3. Collectez des données réelles avec python collect_real_data.py
  4. Lancez l'application Streamlit avec python run_dashboard.py
